## About the Website
This website is simply my little corner of the internet, where I'll display projects I'm working on, write blog posts about what I've done recently, or share cool things I've found elsewhere on the web.
It's a static site built with [Hugo](https://gohugo.io), using locally-sourced handcrafted HTML and CSS and containing no JavaScript. The HTML was checked with the [W3C's validator](https://validator.w3.org/nu/).
Go to the Blogs section to read my writing, to Links to find me elsewhere on the internet, or to Projects to see what I've been creating lately.

## About Me
I'm a student studying Cybersecurity and Mandarin Chinese. Given the chance, I will run Linux on anything. I like using Python, HTML, CSS, Bash, Fish, and sometimes a bit of Java (to get me up in the morning).
